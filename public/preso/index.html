Under the Hood

How identity resolution works

  • Web SDK assigns a device ID to every visitor
    A unique anonymous identifier per browser
  • Behavioral events stream in real time
    Page views, product views, configurations
  • Lead submission sends identity signals
    Email, phone, name matched against CRM contacts
  • Data Cloud reconciliation rules match and merge
    Anonymous device + CRM record = one Unified Individual
  • Unified profile includes full CRM enrichment
    Address, MDM ID, dealer history, and every lead
